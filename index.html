Html-CSS-JS

CSS dili sayesinde HTML internet sayfalarının dilediğimiz gibi renklendirilmesi, dilediğimiz yazı tiplerinin kullanılması ya da sayfada bulunan öğelerin istediğimiz gibi biçimlendirilmesi sağlanmaktadır. HTML bu konuda tek başına yeterli olmaz. HTML ile oluşturulan bir internet sitesi ilk çıkan siyah beyaz bir televizyonu andırırken CSS dili de kullanarak sayfaya hareket ve renk katılır. Son olarak kısaltılışı JS olan JavaScript ise bunlardan farklı olarak oldukça popüler olan bir betik dil olarak karşımıza çıkmaktadır. JS aslında bir bakıma HTML sayfalarına çeşitli fonksiyonlar kazandırmaktadır. JavaScript kullanılarak bir sayfaya fonksiyon ve işlevsellik katılmaktadır.
